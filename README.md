# Mobile Image Dataset Builder
A cross-platform mobile app that helps users quickly build clean image datasets for machine learning by swiping to keep or discard images.

Users search for a topic, review images with simple swipe gestures, and export a curated dataset ready for training models.

## Structure
```
/mobile     → React Native app
/server     → Express API
/web        → Export panel
/database   → SQL schema/migrations
```
